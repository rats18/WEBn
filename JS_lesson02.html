1. 만들 기능: Night & Day 모드

1.1 동작 방식

  • night 버튼 클릭 → 배경 검은색, 글자 흰색
  • day 버튼 클릭 → 원래 색상으로 복원

2. 사용할 기술

2.1 HTML: 버튼 만들기

<input type="button" value="night" onclick="JavaScript코드">

2.2 JavaScript: 페이지 변경하기

document.querySelector('body').style.backgroundColor='black';
document.querySelector('body').style.color='white';

3. 핵심 개념 2가지

  1. 사용자 상호작용: 버튼 클릭 → 반응
  2. HTML 제어: 웹페이지 내용을 실시간으로 변경

4. 실습 체험

Night & Day 버튼 체험해보기

아래 버튼을 클릭해서 실제 동작을 확인해보세요:

팁: 버튼을 클릭하면 페이지 전체의 배경색과 글자색이 바뀝니다. F12로 개발자도구를 열어서 Elements 탭에서 HTML 변화를 관찰해보세요!

5. 확인 방법

F12(개발자도구) → Elements 탭에서 버튼 클릭 시 HTML 변화 관찰

6. 학습 포인트

JavaScript가 HTML과 결합하여 동적인 웹페이지를 만드는 과정을 구체적인 예시를 통해 확인. Night & Day 모드 기능은 앞으로 배울 모든 JavaScript 개념들이 실제로 어떻게 활용되는지 보여주는 완성된 예제.
