1. Boolean이란?

  • 영어로 Boolean (불린)
  • 단 2개의 값만 존재: true, false
  • 참과 거짓을 나타내는 데이터타입

2. Boolean 값 사용법

2.1 기본 값

true      // 참

false     // 거짓

2.2 대소문자 주의

true      // 올바름

True      // 틀림

TRUE      // 틀림

3. 다른 데이터타입과 비교

  • Number: 1, 2, 3, 100, 1000... (무한히 많음)
  • String: "hello", "world", "JavaScript"... (엄청나게 많음)
  • Boolean: true, false (단 2개뿐)

4.1 Boolean vs 문자열 차이

4.2 Boolean 계산

5. Boolean의 특징

5.1 단순함의 힘

  • 값이 단 2개뿐이라 매우 단순
  • 컴퓨터가 빠르게 처리 가능
  • 예/아니오, 켜짐/꺼짐 같은 이분법적 상황에 적합

5.2 실생활 예시

  • 전등 스위치: 켜짐(true) / 꺼짐(false)
  • 로그인 상태: 로그인됨(true) / 로그아웃됨(false)
  • Night 모드: 어두움(true) / 밝음(false)

6. Boolean 활용 미리보기

Boolean은 프로그램이 결정을 내리는데 사용됩니다:

  • 현재 상태가 true라면 → 이런 동작
  • 현재 상태가 false라면 → 저런 동작

7. 핵심 기억사항

  1. 2개 값만: true와 false
  2. 대소문자 주의: 반드시 소문자로
  3. 따옴표 없음: true (Boolean) vs 'true' (문자열)
  4. 단순함의 힘: 복잡한 결정을 단순하게
