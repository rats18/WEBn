1. 반복문이란?

  • 영어로 Loop
  • 조건에 따라 같은 코드를 여러 번 실행
  • 반복적 작업의 절망적 상황에서 구원

2. while문 기본 구조

2.1 문법

while (조건) {

    // 반복 실행할 코드

}

2.2 동작 원리

  1. 조건 확인: true면 실행, false면 종료
  2. 코드 실행: 중괄호 안의 코드 실행
  3. 다시 조건 확인: 1번으로 돌아가서 반복
  4. 종료: 조건이 false가 되면 반복 종료

3. 실제 구현 예제

3.1 3번 반복하는 코드

var i = 0;                  // 카운터 변수

while (i < 3) {              // 조건: i가 3보다 작을 때

    document.write("2<br>");  // 실행할 코드

    document.write("3<br>");  // 실행할 코드

    i = i + 1;               // 카운터 증가

}

3.2 실행 과정 분석

// 1차 실행: i=0, 0<3 (true) → 코드 실행 → i=1

// 2차 실행: i=1, 1<3 (true) → 코드 실행 → i=2

// 3차 실행: i=2, 2<3 (true) → 코드 실행 → i=3

// 4차 확인: i=3, 3<3 (false) → 반복 종료

5. 핵심 구성 요소

5.1 카운터 변수

  • var i = 0: 반복 횟수를 세는 변수
  • 관습적으로 i 사용 (꼭 i가 아니어도 됨)

5.2 조건식

  • i < 3: 언제까지 반복할지 결정
  • true일 때 계속, false일 때 종료

5.3 증감식

  • i = i + 1: 매번 카운터를 증가
  • 반복 종료 조건에 도달하게 하는 핵심

6. 중요한 주의사항

6.1 무한루프 방지

// 위험한 코드 (무한루프)

while (true) {

    document.write("끝나지 않음");

    // i 증가가 없으면 영원히 반복

}

6.2 안전한 코드

  • 반드시 종료 조건 설정
  • 카운터 변수 증가 확인
  • 브라우저 자원 고갈 방지

8. 반복문의 핵심 가치

8.1 문제 해결

  • 반복적 코드 작성 → 효율적 반복문
  • 1억줄 복사붙여넣기 → 몇 줄의 코드
  • 수정의 악몽 → 한 곳만 수정

8.2 제어문으로서의 역할

  • 순차 실행에서 반복 실행으로 확장
  • 조건문과 함께 프로그램 흐름 제어

9. for문 미리보기

9.1 while문과 같은 기능

// while문

var i = 0;

while (i < 3) {

    document.write("hello");

    i = i + 1;

}

// for문 (간결한 표현)

for (var i = 0; i < 3; i = i + 1) {

    document.write("hello");

}

10. 반복문 선택 가이드

10.1 while문

  • 조건이 복잡한 경우
  • 몇 번 반복할지 미리 알 수 없는 경우
  • 무한루프가 필요한 경우

10.2 for문

  • 정확한 반복 횟수를 아는 경우
  • 배열과 함께 사용하는 경우
  • 간결한 코드를 원하는 경우

11. 핵심 기억사항

  1. while (조건): 조건이 true인 동안 반복
  2. 카운터 변수: 반복 횟수를 세는 변수 (보통 i)
  3. 증감식: 매번 카운터를 변경해서 종료 조건 도달
  4. 무한루프 주의: 종료 조건 반드시 확인
