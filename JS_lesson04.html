1. 이번 강의에서 배울 것

  • 사용자와 상호작용하는 핵심 기능: 이벤트(Event)
  • 버튼 클릭, 텍스트 입력 등에 반응하는 방법

2. 기본 구조: 버튼 + 이벤트

2.1 버튼 만들기

<input type="button" value="hi" onclick="alert('hi')">

2.2 동작 과정

  1. 사용자가 버튼 클릭
  2. onclick 속성의 JavaScript 코드 실행
  3. alert('hi') → 경고창 팝업

3. 주요 이벤트들

3.1 클릭 이벤트

<input type="button" value="hi" onclick="alert('안녕')">

3.2 내용 변경 이벤트

<input type="text" onchange="alert('changed')">

→ 텍스트 입력 후 포커스 벗어날 때 실행

3.3 키 누름 이벤트

<input type="text" onkeydown="alert('key down')">

→ 아무 키나 누를 때마다 실행

5. 이벤트 작동 원리

5.1 웹브라우저의 동작

  1. onclick 속성을 만나면 JavaScript 코드를 기억
  2. 사용자가 해당 요소를 클릭할 때까지 대기
  3. 클릭 발생 시 기억하고 있던 코드 실행

5.2 중요한 규칙

  • on으로 시작하는 속성들 = 이벤트 속성
  • 이벤트 속성의 값 = 반드시 JavaScript 코드

6. 실무 활용

6.1 Night & Day 버튼 예제

<input type="button" value="night" onclick="JavaScript코드로 테마 변경">

<input type="button" value="day" onclick="JavaScript코드로 테마 복원">

7. 핵심 개념

  • 이벤트: 웹브라우저에서 일어나는 기념할 만한 사건들
  • 상호작용: 사용자 행동(클릭, 입력)에 반응하는 웹페이지
  • 약 10~20개의 주요 이벤트가 정의되어 있음
