1. 데이터타입이란?

  • 컴퓨터가 데이터를 종류별로 분류하는 방법
  • JavaScript에는 6개의 기본 데이터타입 + 객체가 존재

2. 숫자(Number) 데이터타입

2.1 표현 방법

1        // 숫자 1

1+1      // 계산 결과: 2

2.2 계산 방법

2-1      // 빼기: 1

2*4      // 곱하기: 8

6/2      // 나누기: 3

3. 문자열(String) 데이터타입

3.1 표현 방법

"hello world"    // 큰따옴표

'hello world'    // 작은따옴표

규칙: 큰따옴표로 시작하면 큰따옴표로 끝내기

3.2 주요 기능들

"hello world".length         // 글자 수: 11

"hello world".toUpperCase()  // 대문자 변환: "HELLO WORLD"

"hello world".indexOf('o')     // 문자 위치 찾기: 4

"hello world".indexOf('world')  // 문자열 위치: 6

" hello ".trim()             // 공백 제거: "hello"

4. 중요한 차이점

4.1 숫자 vs 문자열 연산

1+1      // 숫자: 2

"1"+"1"    // 문자열: "11"

6. Night & Day 예제에서

document.querySelector('body').style.backgroundColor='black';

'body', 'black' → 따옴표로 감싸진 문자열

7. 핵심 기억사항

  1. 숫자: 따옴표 없이 쓰고 계산 가능
  2. 문자열: 따옴표로 감싸고 텍스트 처리 가능
  3. 연산 차이: 1+1=2, "1"+"1"="11"
  4. 문자열 기능: length, toUpperCase, indexOf, trim 등
