1. 토글 기능이란?

토글(Toggle)은 하나의 버튼으로 두 가지 상태를 번갈아 가며 전환하는 기능입니다. Night & Day 모드에서 한 번 클릭하면 밤 모드로, 다시 클릭하면 낮 모드로 변경되는 것이 대표적인 예시입니다.

1.1 기본 개념

기존의 Night & Day 버튼은 항상 같은 동작만 수행했습니다. 이제 조건문을 활용하여 현재 상태를 판단하고, 그에 따라 다른 동작을 수행하도록 개선하겠습니다.

// 기존 방식: 항상 같은 동작 function nightDayHandler() { // 항상 밤 모드로만 변경 } // 토글 방식: 상태에 따라 다른 동작 function toggleHandler() { if (현재가_낮_모드라면) { 밤_모드로_변경(); } else { 낮_모드로_변경(); } }

2. 현재 상태 판단하기

토글 기능을 구현하려면 먼저 현재 어떤 상태인지 알아야 합니다. 여러 가지 방법이 있습니다.

2.1 배경색으로 상태 판단

function toggleMode() { if (document.body.style.backgroundColor === 'black') { // 현재 밤 모드 → 낮 모드로 변경 document.body.style.backgroundColor = 'white'; document.body.style.color = 'black'; } else { // 현재 낮 모드 → 밤 모드로 변경 document.body.style.backgroundColor = 'black'; document.body.style.color = 'white'; } }

2.2 버튼 텍스트로 상태 판단

function toggleMode(self) { if (self.value === 'night') { // 현재 낮 모드 → 밤 모드로 변경 document.body.style.backgroundColor = 'black'; document.body.style.color = 'white'; self.value = 'day'; } else { // 현재 밤 모드 → 낮 모드로 변경 document.body.style.backgroundColor = 'white'; document.body.style.color = 'black'; self.value = 'night'; } }

3. 실습: 토글 기능 구현

3.1. 기본 토글 버튼

하나의 버튼으로 배경색과 글자색을 번갈아 변경해보세요.

토글 데모 영역

이 영역의 색상이 변경됩니다.

상태: 현재 낮 모드

3.2. 링크 색상까지 포함한 고급 토글

배경, 글자, 링크 색상을 모두 토글해보세요.

고급 토글 데모

링크도 함께 변경됩니다: 테스트 링크

상태: Day 모드 (링크: 파란색)

3.3. 버튼 텍스트 스타일 포함 토글

버튼의 텍스트와 스타일까지 함께 변경되는 토글을 구현해보세요.

버튼 스타일 토글

버튼의 모양도 함께 변경됩니다.

상태: 낮 모드 (밝은 버튼)

4. 토글 구현 패턴

효과적인 토글 기능을 위한 다양한 구현 패턴을 알아보겠습니다.

4.1 상태 변수 활용

let isDarkMode = false; function toggle() { if (isDarkMode) { // 밤 모드 → 낮 모드 document.body.style.backgroundColor = 'white'; document.body.style.color = 'black'; isDarkMode = false; } else { // 낮 모드 → 밤 모드 document.body.style.backgroundColor = 'black'; document.body.style.color = 'white'; isDarkMode = true; } }

4.2 CSS 클래스 토글

function toggleClass() { document.body.classList.toggle('dark-mode'); // 버튼 텍스트도 함께 변경 const btn = document.querySelector('#toggleBtn'); if (document.body.classList.contains('dark-mode')) { btn.textContent = '☀️ Day'; } else { btn.textContent = '🌙 Night'; } }

5. 토글 기능의 장점

5.1 사용자 경험 개선

• 직관적 인터페이스: 하나의 버튼으로 두 상태를 제어

• 공간 절약: 여러 버튼 대신 하나의 토글 버튼

• 명확한 피드백: 현재 상태를 버튼 텍스트로 표시

5.2 코드 효율성

• 중복 제거: 비슷한 기능을 하나의 함수로 통합

• 유지보수성: 상태 관리가 한 곳에 집중

• 확장성: 새로운 상태 추가가 용이

6. 실무 활용 예시

6.1 메뉴 열기/닫기

function toggleMenu() { const menu = document.querySelector('#sideMenu'); if (menu.style.display === 'none') { menu.style.display = 'block'; document.querySelector('#menuBtn').textContent = '✕ Close'; } else { menu.style.display = 'none'; document.querySelector('#menuBtn').textContent = '☰ Menu'; } }

6.2 좋아요 버튼

function toggleLike(self) { if (self.textContent === '♡') { self.textContent = '❤️'; self.style.color = 'red'; } else { self.textContent = '♡'; self.style.color = 'gray'; } }

7. 조건문의 핵심 역할

토글 기능에서 조건문은 현재 상태를 판단하는 핵심 도구입니다.

판단 기준의 다양성

• 스타일 속성: backgroundColor, color, display 등

• HTML 속성: value, textContent, className 등

• 변수 상태: Boolean, 숫자, 문자열 등

• CSS 클래스: classList.contains() 활용
