1. CSS란?

  • HTML, JavaScript와 완전히 다른 컴퓨터 언어
  • 역할: 디자인과 꾸미기
  • HTML의 style 속성 안에 CSS 코드가 들어감

2. style 속성 사용법

2.1 기본 구조

<h1 style="color:powderblue; background-color:coral;">제목</h1>

2.2 CSS 속성 예제

color: powderblue;              /* 글자색 */

background-color: coral;        /* 배경색 */

font-weight: bold;             /* 글자 굵게 */

3. 실습 체험

4. CSS 속성 찾는 방법

4.1 검색 키워드

  • css background color property
  • css font bold property
  • 원하는 효과 + css + property

4.2 문법 구조

  • 속성명: 값; 형태
  • 세미콜론(;)으로 구분
  • 여러 속성을 나열 가능

5. HTML vs CSS 역할 분담

  • HTML: 구조와 내용
  • CSS: 디자인과 스타일
  • style 속성: HTML과 CSS를 연결하는 다리

6. JavaScript에서 CSS 속성명

6.1 CSS vs JavaScript 차이점

CSS: background-color → JavaScript: backgroundColor

CSS: font-weight → JavaScript: fontWeight

CSS의 하이픈(-)은 JavaScript에서 카멜케이스로 변환

7. 핵심 기억사항

  1. style 속성: HTML 태그에 CSS 적용하는 방법
  2. CSS 문법: 속성명:값; 형태
  3. 검색 활용: 원하는 효과를 찾아서 적용
  4. JavaScript 사용시: 하이픈을 카멜케이스로 변경
