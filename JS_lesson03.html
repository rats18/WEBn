1. 이번 강의에서 배울 것

  • HTML 문서에 JavaScript 코드 넣는 방법
  • <script> 태그 사용법

2. JavaScript 삽입 방법

2.1 기본 구조

<!DOCTYPE html>
<html>
<head>
    <title>ex1</title>
</head>
<body>
    <script>
        JavaScript 코드가 여기 들어감
    </script>
</body>
</html>

2.2 핵심 포인트

  • <script> 태그 안에 JavaScript 코드 작성
  • 웹브라우저가 <script> 태그 안의 코드를 JavaScript로 해석

3. 실습 예제

3.1 텍스트 출력

document.write('hello world');

→ 화면에 "hello world" 출력

3.2 계산 기능

document.write(1+1);

→ 화면에 "2" 출력

4. HTML vs JavaScript 차이점

구분 HTML JavaScript
성격 정적 동적
계산 1+1 → 문자 그대로 "1+1" 1+1 → 계산 결과 "2"
기능 내용 표시만 계산, 처리 가능

6. 핵심 기억사항

  1. <script> 태그: JavaScript 코드임을 브라우저에 알려줌
  2. document.write(): 웹페이지에 글씨 출력하는 명령어
  3. 동적 처리: JavaScript는 계산기처럼 동작 가능
