1. 비교연산자란?

  • 두 값을 비교해서 true 또는 false를 결과로 내는 연산자
  • Boolean 값을 만들어내는 도구

2. 주요 비교연산자

2.1 동등 비교 연산자 (===)

1 === 1    // true (같음)

1 === 2    // false (다름)

"hello" === "hello"    // true

"hello" === "world"    // false

2.2 크기 비교 연산자

1 < 2      // true (1이 2보다 작음)

1 > 1      // false (1이 1보다 크지 않음)

2 >= 2     // true (2가 2보다 크거나 같음)

1 <= 0     // false (1이 0보다 작거나 같지 않음)

3. if문 기본 구조

3.1 문법

if (조건) {

    // 조건이 true일 때 실행되는 코드

} else {

    // 조건이 false일 때 실행되는 코드

}

3.2 동작 원리

  1. if 괄호 안의 조건을 확인
  2. 결과가 true면 if 블록 실행
  3. 결과가 false면 else 블록 실행

4.2 크기 비교 체험

5. 조건문의 힘

5.1 프로그램 흐름 제어

  • 기존: 위에서 아래로 순차 실행
  • 조건문: 상황에 따라 다른 코드 실행

5.2 실생활 예시

현재 시간이 밤이면 → 어두운 테마

현재 시간이 낮이면 → 밝은 테마

6. 비교연산자 종류 정리

  • ===: 같다
  • !==: 다르다
  • >: 크다
  • <: 작다
  • >=: 크거나 같다
  • <=: 작거나 같다

7. 핵심 기억사항

  1. 비교연산자: 두 값을 비교해서 true/false 생성
  2. if문: Boolean 값에 따라 실행 코드 결정
  3. 프로그램 흐름 제어: 순차 실행에서 조건부 실행으로 진화

8. Night & Day 예제에서 활용

현재 상태를 비교해서 Night인지 Day인지 판단하고, 그에 따라 다른 동작을 수행할 수 있음. 하나의 버튼으로 두 가지 모드를 전환하는 토글 기능 구현이 가능해짐.
