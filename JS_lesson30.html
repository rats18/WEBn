30강: 속성 제어

목표: JavaScript로 HTML 속성을 읽고 변경하는 방법 학습

키워드: getAttribute, setAttribute, removeAttribute, 속성 제어

🎯 이번 강의의 핵심
HTML 요소의 속성(attribute)을 JavaScript로 동적으로 제어하여 더욱 풍부한 사용자 상호작용을 구현합니다.

1. HTML 속성(Attribute) 이해

HTML 요소는 다양한 속성을 가지며, JavaScript로 이를 읽고 변경할 수 있습니다.

1.1 주요 HTML 속성들

<img src="image.jpg" alt="설명" width="300"> <a href="https://example.com" target="_blank">링크</a> <input type="text" value="기본값" placeholder="입력하세요"> <div class="container" id="main">내용</div>

2. 속성 제어 메서드

2.1 getAttribute() - 속성 값 읽기

var element = document.querySelector('#myImage'); var imageSrc = element.getAttribute('src'); var imageAlt = element.getAttribute('alt'); console.log('이미지 경로:', imageSrc); console.log('대체 텍스트:', imageAlt);

2.2 setAttribute() - 속성 값 설정

var element = document.querySelector('#myImage'); element.setAttribute('src', 'new-image.jpg'); element.setAttribute('alt', '새로운 이미지'); element.setAttribute('width', '400');

2.3 removeAttribute() - 속성 제거

var element = document.querySelector('#myImage'); element.removeAttribute('width'); // width 속성 제거 element.removeAttribute('height'); // height 속성 제거

3.1 href 속성 동적 변경

6. 속성 vs 프로퍼티 차이점

구분 HTML 속성 (Attribute) DOM 프로퍼티 (Property)
접근 방법 getAttribute(), setAttribute() element.property 형태
데이터 타입 항상 문자열 실제 데이터 타입
예시 element.getAttribute('checked') element.checked
HTML 반영 HTML 코드에 바로 반영 DOM에서만 변경

6.1 실제 사용 예시

// 속성 방식 (문자열) element.setAttribute('checked', 'true'); var isChecked = element.getAttribute('checked'); // "true" (문자열) // 프로퍼티 방식 (불린) element.checked = true; var isChecked = element.checked; // true (불린값)
