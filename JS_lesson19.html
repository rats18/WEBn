1. 환상의 콤비: 배열 + 반복문

  • 배열: 연관된 데이터 정리정돈
  • 반복문: 자동화된 순차 처리
  • 결합 시: 폭발적인 시너지 효과

2. 기본 구현 과정

2.1 배열 생성

var coworkers = ['egoing', 'leezche', 'duru', 'taeho'];

2.2 반복문으로 배열 순회

var i = 0;

while (i < coworkers.length) {

    document.write('<li>' + coworkers[i] + '</li>');

    i = i + 1;

}

2.3 핵심 연결점

  • i 값 = 배열 인덱스: i가 0,1,2,3으로 증가
  • 배열 접근: coworkers[i]로 순차적 데이터 접근
  • 동적 길이: coworkers.length로 배열 크기에 맞춰 반복

3. 실습 체험

3.1 기본 배열 순회

3.2 동적 길이 체험

3.3 링크 목록 자동 생성

핵심: 배열의 각 이름으로 링크 주소를 자동 생성. 새로운 동료 추가 시 코드 수정 없이 자동 반영!

4. 실용적 활용 예제

4.1 HTML 목록 자동 생성

var coworkers = ['egoing', 'leezche', 'duru', 'taeho'];

var i = 0;

while (i < coworkers.length) {

    document.write('<li><a href="http://a.com/' + coworkers[i] + '">' + coworkers[i] + '</a></li>');

    i = i + 1;

}

4.2 결과

  • 각 이름이 링크가 되어 목록 생성
  • 링크 주소: http://a.com/이름
  • 데이터만 변경하면 전체 구조 자동 업데이트

5. 폭발적 효과

5.1 유연성 (Flexibility)

// 멤버 추가

coworkers.push('graphittie');

// → 자동으로 목록에 추가됨 (코드 수정 불필요)

5.2 확장성 (Scalability)

  • 1개든 1억개든 동일한 코드로 처리
  • 데이터 양에 관계없이 일관된 동작

5.3 유지보수성 (Maintainability)

// HTML 구조 변경이 필요하면

// 반복문 안의 템플릿만 수정

document.write('<div class="member">' + coworkers[i] + '</div>');

// → 모든 항목에 일괄 적용

6. 핵심 패턴

6.1 동적 길이 처리

// 고정된 숫자 사용 (나쁜 예)

while (i < 4) { ... }

// 동적 길이 사용 (좋은 예)

while (i < coworkers.length) { ... }

6.2 인덱스 활용

  • i 변수가 배열 인덱스와 완벽 매칭
  • 반복문의 카운터 = 배열의 순서
  • 우연이 아닌 필연적 설계

7. 실제 활용 가치

7.1 데이터 중심 프로그래밍

  • 로직 고정, 데이터 변경으로 결과 생성
  • 프로그래머는 데이터에만 집중 가능
  • 유연하고 탄력적인 코드 구조

7.2 웹 개발에서의 응용

  • 동적 메뉴 생성
  • 사용자 목록 표시
  • 게시물 목록 렌더링
  • 모든 반복적 UI 생성

8. 배열과 반복문의 완벽한 조화

8.1 인덱스 카운터 일치

var i = 0;              // 반복문 카운터

coworkers[i];         // 배열 인덱스

// i가 0→1→2→3 증가하며 배열의 모든 요소에 순차 접근

8.2 자동 종료 조건

while (i < coworkers.length)

// 배열 크기만큼 정확히 반복하고 자동 종료

9. 배열과 반복문의 실제 활용

9.1 성적 계산 시스템

9.2 메뉴 시스템

핵심: 반복문으로 두 배열(메뉴, 가격)을 동시에 처리하여 완전한 정보 제공

10. 핵심 기억사항

  1. i 변수: 반복문 카운터이자 배열 인덱스
  2. length 속성: 배열 크기에 맞춘 자동 반복 종료
  3. 동적 처리: 데이터만 바꾸면 전체 결과 변경
  4. 확장성: 1개든 1억개든 동일한 코드로 처리
