1. 변수란?

  • 바뀔 수 있는 값을 저장하는 공간
  • 영어로 Variable (vs 상수 Constant)

2. 기본 사용법

2.1 변수 선언과 대입

var x = 1;        // x라는 변수에 1을 대입

var y = 1;        // y라는 변수에 1을 대입

x + y;            // 결과: 2

x = 1000;         // x의 값을 1000으로 변경

x + y;            // 결과: 1001

2.2 대입 연산자 =

  • 오른쪽 값을 왼쪽 변수에 대입하는 연산자
  • 1 = 2 → 에러 (상수에는 대입 불가)

3. 변수 사용의 효용성

3.1 문제 상황: 반복되는 이름 변경

// 비효율적인 방법 - egoing이 여러 곳에 하드코딩

alert("Hello egoing. Lorem ipsum dolor sit amet, egoing consectetur adipisicing elit. egoing");

3.2 해결책: 변수 활용

var name = "egoing";

alert("Hello " + name + ". Lorem ipsum dolor sit amet, " + name + " consectetur adipisicing elit. " + name);

// 이름 변경이 필요할 때

name = "leezche";  // 한 곳만 수정하면 모든 곳이 변경됨

4. 실습 체험

5. 핵심 개념

5.1 Variable vs Constant

  • Variable (변수): 값이 바뀔 수 있는 것
  • Constant (상수): 값이 바뀌지 않는 것 (예: 숫자 1, 2, 3...)

5.2 var 키워드

var name = "egoing";  // 권장하는 방법

name = "egoing";      // 가능하지만 var 사용을 권장

6. Night & Day 예제에서

document.querySelector('body').style.backgroundColor='black';

  • = → 대입 연산자
  • 'black' 값이 backgroundColor 변수에 대입됨

7. 중요한 마인드셋

극단적으로 생각하기: 3개가 아니라 1억개라고 가정. 한 곳만 수정하면 모든 곳이 변경되는 효율성이 변수의 핵심 가치.
