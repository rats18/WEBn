1. 콘솔이란?

  • 파일 없이 JavaScript 코드를 즉석에서 실행할 수 있는 도구
  • 계산기처럼 가벼운 작업에 활용

2. 콘솔 사용 방법

2.1 열기

  1. 웹페이지에서 오른쪽 클릭 → 검사
  2. Console 탭 선택
  3. 깜빡이는 커서에서 JavaScript 코드 입력

2.2 단축키

ESC: Elements 탭에서 하단에 Console 창 토글

3. 실용적 활용 예제

3.1 문자 개수 세기

"복사한 긴 텍스트".length

→ 문자열의 글자 수 출력

3.2 경고창으로 결과 보기

alert("복사한 긴 텍스트".length);

3.3 웹페이지 제어

  • 콘솔에서 실행하는 JavaScript는 현재 보고 있는 웹페이지를 대상으로 동작
  • 이미 만들어진 웹사이트를 나의 필요에 따라 제어 가능

4. 콘솔 실습 가이드

지금 이 페이지에서 콘솔을 열고 다음 명령들을 실제로 입력해보세요:

4.1 기본 계산

1+1

10*5

100/4

4.2 문자열 조작

"안녕하세요".length

"JavaScript".toUpperCase()

4.3 이 페이지 제어하기

document.querySelector('h1').style.color='red'

→ 이 페이지의 제목이 빨간색으로 변경됩니다!

4.4 경고창 띄우기

alert('콘솔에서 안녕!')

5. 실무 활용 사례

5.1 랜덤 추첨

댓글에서 랜덤하게 당첨자 선정

5.2 데이터 처리

웹페이지의 정보를 즉석에서 가공

5.3 간단한 계산

복잡한 수식을 빠르게 처리

6. 콘솔 사용 요령

6.1 콘솔 열기 단계별 안내

  1. 이 페이지에서 마우스 오른쪽 클릭
  2. "검사" 또는 "개발자 도구" 클릭
  3. 개발자 도구 창이 열리면 "Console" 탭 클릭
  4. 커서가 깜빡이는 곳에 JavaScript 코드 입력
  5. Enter 키로 실행

6.2 주의사항

  • 오타 주의: JavaScript는 대소문자를 구분함
  • 세미콜론(;)은 생략 가능하지만 써주는 것이 좋음
  • 에러가 나면 빨간 글씨로 표시됨

7. 핵심 개념

  • 즉석 실행: 파일 생성 없이 코드 테스트
  • 현재 페이지 대상: 보고 있는 웹페이지에서 동작
  • 실생활 문제 해결: 거대한 개발이 아닌 일상의 작은 문제 해결

8. 실습 미션

콘솔을 열고 다음 미션을 수행해보세요:

  1. 자신의 이름 글자 수 세기
  2. 간단한 수학 계산하기
  3. 이 페이지의 제목 색깔 바꾸기
  4. 경고창으로 메시지 출력하기

팁: 콘솔은 JavaScript 학습의 든든한 동반자입니다. 궁금한 것이 있으면 언제든 콘솔에서 실험해보세요!
