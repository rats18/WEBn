29강: CSS 변경

목표: JavaScript로 CSS 스타일을 동적으로 변경하는 고급 기법 마스터

키워드: style 속성, 동적 스타일링, CSS 프로퍼티, 카멜케이스

🎯 이번 강의의 핵심
JavaScript를 사용해 웹페이지의 CSS 스타일을 실시간으로 변경하여 동적이고 상호작용하는 사용자 인터페이스를 구현합니다.

1. CSS와 JavaScript의 만남

28강에서 배운 DOM 제어를 확장하여 더욱 다양한 CSS 속성을 동적으로 조작합니다.

1.1 CSS 속성 접근 방법

// 기본 패턴: element.style.property = value document.querySelector('body').style.backgroundColor = 'black'; document.querySelector('body').style.color = 'white'; document.querySelector('body').style.fontSize = '18px';

1.2 CSS vs JavaScript 속성명 차이

2. 다양한 CSS 속성 제어

2.1 색상 관련 속성

// 색상 지정 방법들 element.style.color = 'red'; // 색상명 element.style.color = '#3498db'; // 16진수 element.style.color = 'rgb(52, 152, 219)'; // RGB element.style.color = 'rgba(52, 152, 219, 0.5)'; // RGBA (투명도)

2.2 크기와 간격 속성

// 크기와 간격 element.style.width = '200px'; element.style.height = '100px'; element.style.margin = '10px'; element.style.padding = '20px'; element.style.fontSize = '16px';

2.3 레이아웃 속성

// 레이아웃과 위치 element.style.display = 'block'; // 또는 'none', 'inline' element.style.position = 'absolute'; element.style.top = '50px'; element.style.left = '100px';

3. 조건부 스타일링

현재 상태에 따라 다른 스타일을 적용하는 고급 패턴입니다.

3.1 토글 패턴

function toggleTheme(element) { if(element.style.backgroundColor === 'black') { element.style.backgroundColor = 'white'; element.style.color = 'black'; } else { element.style.backgroundColor = 'black'; element.style.color = 'white'; } }

3.2 다중 상태 관리

function setTheme(themeName) { var body = document.querySelector('body'); if(themeName === 'dark') { body.style.backgroundColor = 'black'; body.style.color = 'white'; } else if(themeName === 'blue') { body.style.backgroundColor = 'navy'; body.style.color = 'lightblue'; } else if(themeName === 'green') { body.style.backgroundColor = 'darkgreen'; body.style.color = 'lightgreen'; } }

4. 다중 요소 스타일링

여러 요소를 한 번에 스타일링하는 효율적인 방법입니다.

4.1 querySelectorAll 활용

// 모든 링크의 색상을 한 번에 변경 var links = document.querySelectorAll('a'); for(var i = 0; i < links.length; i++) { links[i].style.color = 'red'; links[i].style.textDecoration = 'none'; }

4.2 클래스별 스타일링

// 특정 클래스의 모든 요소 스타일링 var buttons = document.querySelectorAll('.demo-button'); for(var i = 0; i < buttons.length; i++) { buttons[i].style.backgroundColor = 'blue'; buttons[i].style.color = 'white'; buttons[i].style.border = 'none'; }

5. 실무에서 자주 사용되는 CSS 제어 패턴

5.1 표시/숨김 제어

// 요소 숨기기/보이기 element.style.display = 'none'; // 숨기기 element.style.display = 'block'; // 보이기 element.style.visibility = 'hidden'; // 자리 차지하며 숨기기 element.style.visibility = 'visible'; // 보이기

5.2 투명도 제어

// 투명도 조절 (0: 완전 투명, 1: 완전 불투명) element.style.opacity = '0.5'; // 반투명 element.style.opacity = '0'; // 투명 element.style.opacity = '1'; // 불투명

5.3 전환 효과

// CSS 전환 효과 추가 element.style.transition = 'all 0.3s ease'; element.style.transform = 'scale(1.2)'; // 1.2배 확대 element.style.transform = 'rotate(45deg)'; // 45도 회전
