1. 목표

  • Night 버튼 클릭 시 웹페이지를 검은색으로 변경
  • JavaScript로 HTML 요소를 동적으로 제어하는 방법 학습

2. HTML + CSS 기본 구조

2.1 CSS로 직접 스타일 적용

<body style="background-color:black; color:white">

  • style 속성에는 CSS 언어가 와야 함
  • CSS의 역할: 디자인과 꾸미기

2.2 문제점

  • HTML은 정적 언어: 한번 화면에 표시되면 자기를 바꿀 능력이 없음
  • 사용자 상호작용으로 동적 변경 불가

3. JavaScript의 해결책

3.1 동적 스타일 변경

// Night 버튼 클릭 시 실행되는 코드

document.querySelector('body').style.backgroundColor='black';

document.querySelector('body').style.color='white';

3.2 동작 과정

  1. 사용자가 Night 버튼 클릭
  2. JavaScript 코드 실행
  3. body 태그의 style 속성이 실시간으로 추가/변경
  4. 웹페이지가 즉시 어두운 테마로 변경

4. 실습 체험

5. 필요한 지식 요소

5.1 CSS 기초

  • background-color: 배경색 설정
  • color: 글자색 설정
  • CSS 문법과 속성들

5.2 JavaScript로 태그 선택

  • document.querySelector('body'): body 태그 선택
  • 제어하고자 하는 HTML 요소를 정확히 찾는 방법

6. 핵심 개념

6.1 HTML vs JavaScript의 차이

  • HTML: 정적, 한번 화면에 표시되면 변경 불가
  • JavaScript: 동적, 실시간으로 HTML 요소 제어 가능

6.2 웹브라우저 제어의 원리

  1. JavaScript가 HTML 태그를 선택
  2. 선택한 태그의 속성을 변경
  3. 웹브라우저가 변경사항을 즉시 반영
