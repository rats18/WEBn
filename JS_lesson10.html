2. HTML 버튼 기본 구조

2.1 버튼 생성

<input type="button" value="night" onclick="JavaScript코드">

<input type="button" value="day" onclick="JavaScript코드">

3. JavaScript로 태그 선택하기

3.1 document.querySelector() 사용법

document.querySelector('body')

3.2 선택자 종류

  • 태그 선택: document.querySelector('body')
  • ID 선택: document.querySelector('#target')
  • 클래스 선택: document.querySelector('.myclass')

4. 스타일 속성 변경하기

4.1 배경색 변경

document.querySelector('body').style.backgroundColor = 'black';

4.2 글자색 변경

document.querySelector('body').style.color = 'white';

5. CSS vs JavaScript 속성명

CSS: background-color → JavaScript: backgroundColor

CSS: color → JavaScript: color

중요: CSS의 하이픈(-)은 JavaScript에서 카멜케이스로 변환

  3. 이벤트 연결: onclick으로 사용자 상호작용 구현
