1. 배열이란?

  • 서로 연관된 데이터를 정리정돈해서 담아두는 수납상자
  • 영어로 Array
  • 데이터가 많아질수록 필수적인 정리 도구

2. 배열 기본 문법

2.1 배열 생성

var coworkers = ['egoing', 'leezche'];

  • 대괄호 []로 시작하고 끝남
  • 값들을 쉼표 ,로 구분
  • 변수에 담아서 사용

2.2 데이터 가져오기 (인덱스 사용)

coworkers[0]    // 'egoing' (첫 번째)

coworkers[1]    // 'leezche' (두 번째)

  • 인덱스는 0부터 시작
  • 배열이름[인덱스번호] 형태

2.3 배열 길이 확인

coworkers.length    // 2 (원소 개수)

2.4 데이터 추가

coworkers.push('duru');     // 끝에 추가

coworkers.push('taeho');    // 끝에 추가

coworkers.length;           // 4개로 증가

4. 핵심 개념

4.1 인덱스 (Index)

  • 배열에서 각 데이터의 위치 번호
  • 0부터 시작: 첫 번째가 0번, 두 번째가 1번...

4.2 배열의 비유

  • 집의 수납상자와 같은 개념
  • 물건이 많아지면 → 수납상자 필요
  • 데이터가 많아지면 → 배열 필요

5. 다양한 배열 기능

5.1 검색으로 기능 찾기

  • javascript array 검색
  • push, pop, concat, join 등 다양한 메서드
  • 검색 능력이 암기보다 중요

5.2 실용적 활용

// 데이터 추가

coworkers.push('새멤버');

// 개수 확인

coworkers.length;

// 특정 위치 데이터 가져오기

coworkers[2];

6. 배열의 실용성

6.1 데이터 관리의 혁신

// 배열 없이 (비효율적)

var member1 = 'egoing';

var member2 = 'leezche';

var member3 = 'duru';

// 배열 사용 (효율적)

var members = ['egoing', 'leezche', 'duru'];

6.2 확장성

  • 멤버가 1명이든 1000명이든 동일한 방식으로 관리
  • 새로운 멤버 추가가 매우 간단
  • 전체 멤버 수를 즉시 확인 가능

7. 배열 문법 정리

7.1 생성

var 배열명 = [값1, 값2, 값3];

7.2 접근

배열명[인덱스]    // 특정 값 가져오기

7.3 길이

배열명.length     // 배열 크기

7.4 추가

배열명.push(새값);  // 끝에 추가

8. 핵심 기억사항

  1. 배열 = 데이터 수납상자
  2. 인덱스는 0부터 시작
  3. .length로 개수 확인
  4. .push()로 데이터 추가
  5. 검색으로 다양한 기능 활용

10. 실생활 비유

친구들의 연락처를 관리한다면:

  • 배열 없이: 친구1, 친구2, 친구3... 변수를 개별 관리
  • 배열 사용: friends = ['철수', '영희', '민수'] 한 번에 관리

새로운 친구가 생기면 friends.push('새친구')로 간단히 추가!
